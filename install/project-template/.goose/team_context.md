# Team Context

## Project: Task Tracker
A simple command-line task tracking application used for Goose training exercises.

## Tech Stack
- Python 3.10+
- No external dependencies (stdlib only)
- pytest for testing

## Commands
- Run app: `python -m src.cli`
- Run tests: `python -m pytest tests/ -v`
- Run single test: `python -m pytest tests/test_tasks.py -v`

## Conventions
- Source code in `src/`
- Tests in `tests/`, mirroring src structure
- Functions use snake_case, classes use PascalCase
- No type annotations required (but welcome)
